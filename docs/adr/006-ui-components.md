# ADR-006: UIコンポーネントライブラリ

## ステータス

承認済み

## コンテキスト

住民向け（シンプル・大きなUI）と管理者向け（テーブル・フォーム中心）の2つのUI要件がある。一貫性のあるコンポーネントを効率的に構築したい。

## 決定

**shadcn/ui** を採用する。

初期化コマンド:
```bash
pnpm dlx shadcn@latest init --preset b2D30ZsUV --template next --monorepo
```

## 理由

- コンポーネントがプロジェクトにコピーされるため、カスタマイズの自由度が高い
- Tailwind CSS + Radix UI ベースで、アクセシビリティが確保されている
- 住民向けの大きなボタン・カードと管理者向けのテーブル・フォームを同じ基盤で実現できる
- `--preset b2D30ZsUV` でブランドカラーに近いプリセットを使用し、デザインシステムのカラーに合わせてカスタマイズ
- `--monorepo` フラグでpnpm workspace構成に対応

## コンポーネント方針

- shadcn/uiのコンポーネントをベースに、住民向け・管理者向けのバリアントを追加
- 住民向け: サイズを大きく、角丸を大きく、フォントを太く
- 管理者向け: shadcn/uiのデフォルトサイズをそのまま活用
