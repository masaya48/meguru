# SPEC-005: ランディングページ

## 概要

自治会の役員（意思決定者）にめぐるを知ってもらい、無料トライアルまたは問い合わせに誘導するLP。

## ターゲット

- 町内会・自治会の会長・副会長・班長
- 紙の回覧板に課題を感じている人
- ITに詳しくない人が多い → 温かみ・安心感のあるトーン

## URL

`/lp` — 認証不要のパブリックページ

## ページ構成

### 1. ヒーロー

- キャッチコピー: 「紙の回覧板、もう卒業しませんか？」
- サブコピー: 「めぐるは、町内会のお知らせ配信・既読管理・出欠確認をスマホひとつで実現するサービスです」
- ロゴ表示
- スマホモックアップ（CSS/HTMLで構築、回覧一覧画面のイメージ）
- CTA2つ: 「無料で始める」(primary) + 「まずは相談する」(outline)

### 2. 課題提起

- 見出し: 「こんなお悩みありませんか？」
- 3つの課題カード:
  - 「回覧板が回るのに1週間以上かかる」
  - 「誰が読んだか把握できない」
  - 「出欠確認の集計が手作業で大変」

### 3. 解決策

- 見出し: 「めぐるなら、こう変わります」
- 3つのメリット（アイコン + 見出し + 説明）:
  - 「一斉配信で即日届く」— スマホに通知、LINEでも届く
  - 「既読率がひと目でわかる」— 誰が読んだかリアルタイム確認
  - 「出欠もワンタップ」— ⭕❌の大きなボタンで回答完了

### 4. 機能紹介

- 3枚の機能カード:
  - 回覧配信: お知らせ・出欠確認・アンケートを簡単作成
  - 既読・回答管理: 既読率・回答率をダッシュボードで確認
  - LINE通知: 新着回覧をLINEでお知らせ、出欠もLINEで回答

### 5. 使い方

- 見出し: 「かんたん3ステップ」
- ステップ1: 「自治会を登録する」— 町内会名を入力して無料アカウント作成
- ステップ2: 「住民を招待する」— QRコードやLINEで住民を招待
- ステップ3: 「回覧を配信する」— タイトルと本文を書いて配信ボタン

### 6. よくある質問 (FAQ)

- Q: 料金はかかりますか？ → 現在は無料でお使いいただけます
- Q: スマホが苦手な人でも使えますか？ → 大きな文字とシンプルな画面で、高齢者の方にも安心
- Q: 個人情報は大丈夫ですか？ → データは暗号化して安全に管理
- Q: パソコンでも使えますか？ → 管理者向けのPC画面も用意しています
- Q: LINEがないと使えませんか？ → LINEなしでもWebブラウザから利用可能

### 7. CTA（再掲）

- 見出し: 「まずは無料で試してみませんか？」
- CTA2つ: 「無料で始める」+ 「まずは相談する」

### 8. 問い合わせフォーム

- id="contact" でページ内リンク
- フィールド: 名前 / 自治会名 / メールアドレス / メッセージ
- Server Actionでメール送信（MailHog対応）
- 送信後: 「お問い合わせありがとうございます」メッセージ表示

### 9. フッター

- ロゴ + 「地域をつなぐ、情報がめぐる」
- © 2026 めぐる

## デザイン

- 既存デザインシステム準拠（brand green, coral, ink）
- 背景: 白ベース、セクション交互にgray-50
- CTA: brand-800（primary）, outline（secondary）
- 温かみのあるトーン、大きな余白
- レスポンシブ（モバイルファースト）
- アニメーションなし（シンプルに）

## 技術

- `apps/web/app/lp/page.tsx` — 単一ページコンポーネント
- `apps/web/app/lp/actions.ts` — 問い合わせフォームServer Action
- 認証不要（レイアウトグループ外に配置）
- OGP metadataを設定
