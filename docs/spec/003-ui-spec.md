# SPEC-003: UI仕様

## デザイン原則

1. **シンプルさ最優先** — 高齢者が迷わず操作できること
2. **大きな文字・大きなボタン** — フォント18px以上、タップ領域は十分に確保
3. **視覚的フィードバック** — 未読/既読、回答済み/未回答を色で明確に区別
4. **最小限の操作ステップ** — 目的の操作に3タップ以内で到達

## 住民向け画面

### 回覧一覧（トップページ）

- **レイアウト**: 縦方向カード型リスト（1カラム）
- **ヘッダー**: 自治会名（青背景・白文字・中央揃え）
- **カード構成**:
  - 未読: 薄黄背景 + 左ボーダー黄色
    - 日付ラベル（🔴 未読 ・ 日付）
    - タイトル（18px・太字）
    - 種類の説明テキスト
    - 「回答する」ボタン（回答が必要な場合、赤背景・白文字）
  - 既読: グレー背景
    - 日付ラベル（✅ 既読 ・ 日付）
    - タイトル（18px・太字）
    - 回答状態テキスト
- **並び順**: 未読を上部に表示、既読は日付降順

### 回覧詳細

- **ナビ**: ← 戻るボタン + 「回覧詳細」テキスト
- **コンテンツ**:
  - タイトル（22px・太字）
  - 投稿者・日付（14px・グレー）
  - 本文（17px・行間1.8・グレー背景パネル）
- **回答セクション**（回答が必要な場合）:
  - 青背景パネル
  - 質問テキスト（18px・太字）
  - 大きなボタン2つ横並び（⭕ 参加する / ❌ 不参加）
  - 回答期限テキスト
- **複数質問時**: 1問ずつ画面切替（ステップ形式）
  - 進捗表示（例: 1/3）
  - 戻る/次へボタン
  - 最終問で「送信する」ボタン

### 設定画面

- LINE連携ボタン
- プロフィール編集（名前）
- ログアウト

## 管理者向け画面

### レイアウト

- **サイドバー**（幅200px・ダークブルー背景）
  - 自治会名
  - メニュー: 回覧一覧 / 新規作成 / 住民管理 / グループ / テンプレート / 設定
  - アクティブ項目: 青背景
- **メインエリア**（グレー背景）
  - ページタイトル
  - コンテンツ

### ダッシュボード / 回覧一覧

- **サマリーカード**（横3列）
  - 配信中の回覧数
  - 未回答者数
  - 今月の配信数
- **回覧テーブル**
  - カラム: タイトル / 種類 / 既読率 / 回答率 / 状態
  - 種類バッジ: 出欠（青）、お知らせ（紫）、アンケート（緑）
  - 状態バッジ: 配信中（緑）、下書き（グレー）、終了（グレー）

### 回覧作成

- タイトル入力
- 本文エディタ（リッチテキスト）
- 種類選択（お知らせ / 出欠確認 / アンケート）
- 配信先選択（全員 / グループ指定）
- 質問追加（出欠・アンケート時）
- 回答期限設定
- テンプレートから作成ボタン
- プレビュー → 配信ボタン

### 住民管理

- 住民テーブル（名前 / グループ / 状態 / LINE連携）
- 招待ボタン（メール / LINE / QRコード）
- 申請承認/却下
- グループ割り当て

## レスポンシブ対応

- 住民向け: モバイルファースト（〜768px）
- 管理者向け: デスクトップ優先（768px〜）、モバイルではサイドバーがハンバーガーメニュー化
